# YouTube Channel Performance Analytics Dashboard

## Overview

A comprehensive Streamlit-based analytics dashboard designed to analyze YouTube channel performance, audience engagement, publishing trends, and estimated revenue insights. The dashboard provides interactive visualizations and deep-dive analytics to help understand content performance and channel growth patterns.

## Features

### Main Dashboard

* Channel Performance Overview
* Interactive Data Visualizations
* Audience Engagement Analysis
* Revenue Insights
* Content Performance Tracking

### Deep Dive Analytics

* Publishing Time Analysis
* Revenue Trends by Weekday
* Video Performance Metrics
* Channel Growth Insights
* Advanced Statistical Analysis

### Interactive Capabilities

* Real-Time Dashboard Updates
* Multi-Page Navigation
* Dynamic Data Exploration
* Interactive Charts and Graphs

## Technologies Used

## Installation

### Clone Repository

```bash
git clone https://github.com/grishu21/YouTube-Channel-Performance-Analytics-Dashboard.git
cd YouTube-Channel-Performance-Analytics-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Dashboard

```bash
cd youtube_dashboard
streamlit run youtube_dashboard.py
```

The application will launch in your browser at:

```text
http://localhost:8501
```
